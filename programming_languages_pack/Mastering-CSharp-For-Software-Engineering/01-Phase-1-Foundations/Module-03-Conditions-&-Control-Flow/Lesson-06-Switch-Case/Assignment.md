# Objective

This assignment will help you practice the **switch case statement** in C# and understand how to use it in real-world menu systems and decision-based programs.

---

# Part 1: Theory Questions

Answer in your own words:

1. What is a switch statement?
2. Why do we use switch instead of multiple if-else statements?
3. What is the purpose of the `break` keyword in switch?
4. What happens if we don’t use `break`?
5. What is the purpose of the `default` case?

# Part 2: Practical Tasks

---

## Task 1: Day of Week System

Create a program that:
- Takes a number (1–7)
- Prints the corresponding day of the week

Example:
- 1 → Monday  
- 2 → Tuesday  
- 3 → Wednesday  
- ...  

---

## Task 2: Grade System

Create a program that:
- Takes a grade character (A, B, C, D, F)
- Prints meaning of each grade

Example:
- A → Excellent  
- B → Good  
- C → Average  
- D → Poor  
- F → Fail  

---

## Task 3: Simple Calculator Menu

Create a switch-based calculator that:
- Takes two numbers
- Uses a menu option:
  - 1 → Add  
  - 2 → Subtract  
  - 3 → Multiply  
  - 4 → Divide  

---

## Task 4: ATM Menu System

Create a program that:
- Uses switch case for menu options:
  - 1 → Deposit  
  - 2 → Withdraw  
  - 3 → Check Balance  
  - 4 → Exit  

---

## Task 5 (Challenge): Role Access System

Create a program that:
- Takes user role as input (Admin, User, Guest)
- Uses switch case to display access level:

Example:
- Admin → Full Access  
- User → Limited Access  
- Guest → View Only
