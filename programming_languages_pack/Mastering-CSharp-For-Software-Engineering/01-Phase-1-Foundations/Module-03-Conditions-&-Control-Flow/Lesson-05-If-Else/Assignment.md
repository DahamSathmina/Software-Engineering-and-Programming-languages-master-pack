# Objective

This assignment will help you practice decision-making in C# using:
- if
- else
- else if
- nested if
- logical operators

# Part 1: Theory Questions

Answer in your own words:

1. What is an if-else statement?
2. What is the difference between if and else if?
3. Why do we use nested if statements?
4. What is the difference between `==` and `=` in conditions?
5. What are logical operators used for?

# Part 2: Practical Tasks

---

## Task 1: Even or Odd Checker

Create a program that:
- Takes a number
- Checks if it is Even or Odd
- Uses `if-else`

---

## Task 2: Positive or Negative Number

Create a program that:
- Takes a number
- Prints whether it is Positive, Negative, or Zero

---

## Task 3: Grade System

Create a program that:
- Takes student marks
- Prints grade using conditions:

- 75+ → A Grade  
- 50–74 → B Grade  
- Below 50 → Fail  

---

## Task 4: Voting Eligibility

Create a program that:
- Takes age input
- Checks if user can vote (18+)

---

## Task 5: Login System (Nested If)

Create a program that:
- Stores username and password
- Checks:
  - If username is correct
  - Then check password
- Print:
  - Login Successful
  - Wrong Password
  - User Not Found

---

## Task 6 (Challenge): ATM System Logic

Create a simple ATM system that:
- Stores account balance
- Allows withdrawal
- Checks:
  - Sufficient balance
  - Invalid amount
  - Successful transaction
