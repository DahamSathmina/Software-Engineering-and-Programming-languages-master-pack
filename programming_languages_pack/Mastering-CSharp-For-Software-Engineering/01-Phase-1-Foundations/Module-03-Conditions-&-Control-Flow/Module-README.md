# Module 03: Conditions & Control Flow
## Mastering C# for Software Engineering (Beginner to Professional)

---

# Overview

In this module, you will learn how to control the flow of a program using **decision-making statements** and **loops**.

Control flow allows your program to:
- Make decisions
- Repeat actions
- Execute different blocks of code based on conditions

---

# Learning Objectives

By the end of this module, you will be able to:

- Use `if`, `else if`, and `else`
- Use `switch` statements
- Understand relational and logical decision making
- Use loops (`for`, `while`, `do-while`)
- Build real-world logic-based programs
