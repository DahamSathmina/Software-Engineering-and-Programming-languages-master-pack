# Module 01: Introduction to C# & .NET
## Mastering C# for Software Engineering (Beginner to Professional)

---

# Overview

This module introduces the fundamentals of **C# programming language** and the **.NET ecosystem**.  
You will understand how modern software is built using Microsoft technologies and how C# fits into real-world development.

---

# Learning Objectives

By the end of this module, you will be able to:

- Understand what C# is and why it is used
- Understand .NET Framework, .NET Core, and .NET 8
- Set up a development environment
- Write and run your first C# program
- Understand the structure of a C# application
- Learn how software is executed in .NET
